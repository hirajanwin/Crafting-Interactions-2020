DESIGN ANALYSIS

What's the time mode

3 modes on the clock

Kids can set the clock by themselves, there are 3 modes: What's the time mode/ Art time mode/ Set my clock mode, which can be switched by rolling the arrow on the top.

1. What's the time mode would light up the key points and the strings in the background.

2. Art time mode provides 3 levels of lighting on each circle pixel for kids to create their own art work! Press the circle pixel for the first time, it would be low light, press the second time would make it become high light!

Set my clock mode





3. Kids can be motivated to get to bed in the night and get up in the morning by setting their own clock. There would be dark light in the background and successful setting sound to guide kids how to set the clock correctly.
